# Nerd Fonts

A VS Code extension for searching, previewing, and inserting Nerd Font icons with ease.

## Features

- **Search Icons**: Find Nerd Font icons by name or unicode hex value
- **Preview**: See icons rendered in real-time before inserting
- **Insert Icons**: Quickly insert Nerd Font icons directly into your code
- **Unicode Hex Support**: Look up icons using their unicode hex codes
- **Fast & Lightweight**: Minimal overhead with instant results

## Usage

1. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Search for "Nerd Fonts" command
3. Type the icon name or unicode hex (e.g., `f47e` or `github`)
4. Preview the icon and press Enter to insert

## Supported Nerd Fonts

- Font Awesome
- Powerline Symbols
- Octicons
- Devicons
- Codicons
- And more...

## Installation

Install from the VS Code Extension Marketplace or manually add this extension to your VS Code extensions folder.
